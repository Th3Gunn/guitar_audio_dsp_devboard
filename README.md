Board made for software development for STM32H723 real-time guitar audio DSP without need to play with unreliable connector cables and breadboards. Includes high impedance (1M) instrumental input going into PCM1808 inputs (with switch to check both L and R ADC inputs) and PCM5102A DAC with 2 stereo outputs that should have possibility to deliver line level signal. There is also connector for UART (to communicate with other MCUs - for example to have dedicated MCU for user interface (buttons, encoders, pots, displays, MIDI, preset switching handling etc.). Input stage has switch that enables unity gain or 2x gain to fit different guitars. Powered from BOSS standard 5.5X2.1 CENTER NEGATIVE!!!! DC jack.
